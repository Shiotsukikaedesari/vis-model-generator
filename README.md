# vis-model-generator

An application that can quickly generate vis-three model configuration sheets.
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### version
```
version 0.1
```
